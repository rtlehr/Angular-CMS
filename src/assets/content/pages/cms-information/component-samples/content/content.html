MainContentComponent Usage Guide

The MainContentComponent is an Angular component designed to dynamically load and display HTML content from an external file. It also supports code highlighting and modal window triggers.

1. Installation & Import

Ensure that you import the component into your Angular module or use it as a standalone component.


import { MainContentComponent } from 'path-to-component/main-content.component';

2. Component Selector

Use the following selector to include the component in your Angular template:


<app-main-content [fileToLoad]="'example.html'" [divId]="'content-area'"></app-main-content>

3. Input Properties

Property Type Description
fileToLoad string The path to the HTML file to load dynamically.
divId string Sets the ID of the container div where content will be injected.

4. Features

  • Dynamic HTML Loading: Loads external HTML files into the component.
  • Code Highlighting: Uses highlight.js to apply syntax highlighting.
  • Modal Triggers: Detects specific HTML elements to open a modal window dynamically.

5. Example Usage

To load an HTML file and enable modal triggering, include the component as shown below:


<app-main-content [fileToLoad]="'content.html'" [divId]="'content-container'"></app-main-content>

6. Modal Triggering

If an element in the loaded HTML has the following structure, the component will open a modal:


<trigger-modal data-content='{"title": "Modal Title", "body": "This is modal content"}'></trigger-modal>

7. Event Handling

You can use the data-click attribute to trigger actions like opening modals dynamically:


<a href="#" data-click="openModalWindow" data-content='{"title": "Dynamic Modal", "body": "Loaded from HTML"}'>Open Modal</a>

8. Error Handling

If the specified HTML file fails to load, the component will display the following fallback message:


<p>Sorry, the content could not be loaded.</p>

Conclusion

The MainContentComponent provides a flexible way to load and display dynamic HTML content with added functionality like modal triggers and code highlighting.
