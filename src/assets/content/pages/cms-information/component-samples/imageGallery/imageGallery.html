Image Gallery Component

This component displays an image gallery with a series of images. Each image is accompanied by a title, description, and a link to learn more. The images are loaded dynamically from a JSON file.

How to Use the Image Gallery Component

Follow these steps to integrate and use the Image Gallery component in your Angular project:

1. Import the Component

First, ensure the Image Gallery component is imported and included in your module (if necessary). You can add it to your componentâ€™s imports section.

<app-image-gallery [fileToLoad]="yourJsonFile" [divId]="yourDivId"></app-image-gallery>

2. Provide JSON Data

Provide a JSON file to load the images. The JSON file should contain an array of objects, each representing an image with the following properties:

  • src: The source URL of the image.
  • title: The title of the image.
  • description: A short description of the image.
  • href: A link to learn more about the image (optional).

3. Example JSON Data

Here is an example of the JSON data used to load the images:

[
  {
    "src": "./assets/images/motorcycle.png",
    "title": "Image 1",
    "description": "This is a short description for Image 1.",
    "href": "https://example.com/image1"
  },
  {
    "src": "./assets/images/snowtruck.jpg",
    "title": "Image 2",
    "description": "This is a short description for Image 2.",
    "href": "https://example.com/image2"
  },
  {
    "src": "./assets/images/flag.jpeg",
    "title": "Image 3",
    "description": "This is a short description for Image 3.",
    "href": "https://example.com/image3"
  }
]

4. The HTML Template

The component uses the <div> structure to display the images in a responsive grid layout. Each image is shown inside a <div> card with its title, description, and an optional link.

<div [id]="divId" class="container my-5">
  <div class="row">
    <div class="col-md-4 mb-4" *ngFor="let image of getImages">
      <div class="card">
        <div class="image-container">
          <img src="{{ image.src }}" class="card-img-top zoom-effect" />
        </div>
        <div class="card-body text-center">
          <h5 class="card-title">{{ image.title }}</h5>
          <p class="card-text">{{ image.description }}</p>
          <a href="{{ image.href }}" class="btn btn-primary" target="_blank">Learn More</a>
        </div>
      </div>
    </div>
  </div>
</div>

5. Example Usage

In your parent component, you can use the Image Gallery component as follows:

<app-image-gallery [fileToLoad]="'images-data.json'" [divId]="'gallery-container'"></app-image-gallery>

Options

  • fileToLoad: The name of the JSON file to load the image data from (e.g., 'images-data.json').
  • divId: The unique ID for the gallery container (e.g., 'gallery-container').

Conclusion

This component will render an image gallery dynamically based on the provided JSON data. The layout is responsive, and each image is displayed in a Bootstrap grid format, making it easy to showcase various images with their titles, descriptions, and optional links.
