FaqComponent Usage Guide

The FaqComponent is an Angular component designed to display a list of frequently asked questions (FAQs). It fetches the questions and answers from a JSON file and provides smooth scrolling to both questions and answers.

1. Installation & Import

Ensure that you import the component into your Angular module or use it as a standalone component.


import { FaqComponent } from 'path-to-component/faq.component';

2. Component Selector

Use the following selector to include the component in your Angular template:


<app-faq [fileToLoad]="'faq-data.json'" [divId]="'faq-list'"></app-faq>

3. Input Properties

Property Type Description
fileToLoad string The path to the JSON file containing the FAQ data.
divId string Sets the ID of the container div for the FAQ list.

4. Features

  • Smooth Scrolling: Click on a question to scroll smoothly to its corresponding answer.
  • Back to Top: After reading an answer, you can click "Back to Top" to return to the list of questions.
  • Dynamic FAQ Loading: The component loads FAQ data from a JSON file.
  • Responsive Design: The component adjusts its layout according to screen size.

5. Example Usage

To load a JSON file containing FAQ data and display the list, use the following example:


<app-faq [fileToLoad]="'faq-data.json'" [divId]="'faq-list'"></app-faq>

6. JSON Structure

The FAQ data should follow this structure:


[
{
    "id": "1",
    "question": "What is your return policy?",
    "answer": "You can return any item within 30 days of purchase with a receipt."
},
{
    "id": "2",
    "question": "How can I track my order?",
    "answer": "You can track your order by logging into your account and visiting the 'Orders' section."
},
{
    "id": "3",
    "question": "Do you offer gift cards?",
    "answer": "Yes, we offer digital and physical gift cards available for purchase online."
},
{
    "id": "4",
    "question": "What payment methods do you accept?",
    "answer": "We accept major credit cards, PayPal, and Apple Pay."
},
{
    "id": "5",
    "question": "Can I cancel or modify my order?",
    "answer": "Yes, you can cancel or modify your order within 24 hours of placing it by contacting customer support."
}
]

7. FAQ Layout

Each FAQ item consists of a clickable question that scrolls to its answer:


<div class="faq-container">
<div id="faq-questions" class="faq-questions">
    <h2>Frequently Asked Questions</h2>
    <ul class="list-group">
        <li class="list-group-item">
            <a href="javascript:void(0)" (click)="scrollToAnswer(faqs.id)">
                {{ faqs.question }}
            </a>
        </li>
    </ul>
</div>

<div class="faq-answers mt-4">
    <div [id]="'answer-' + faqs.id" class="faq-answer">
        <h3>{{ faqs.question }}</h3>
        <p>{{ faqs.answer }}</p>
        <a href="javascript:void(0)" (click)="scrollToTop()">Back to Top</a>
    </div>
</div>
</div>

8. Error Handling

If the JSON file cannot be loaded, the component will not display any FAQ data, and you may observe an empty FAQ list.

Conclusion

The FaqComponent is a flexible way to display FAQ data from a JSON file, with smooth scrolling functionality to enhance the user experience.
