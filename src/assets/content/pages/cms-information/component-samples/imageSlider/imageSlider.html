Image Slider Component

The Image Slider component displays a set of images in a sliding format, where the images transition automatically at a set interval. Users can also manually navigate between images using the previous and next buttons.

Inputs

  • fileToLoad (string) - The path to the JSON file containing image data. The JSON file should include the image sources, titles, and additional configuration like slider width, height, and interval.
  • sliderWidth (string) - The width of the slider. You can set it as a percentage (e.g., '100%') or fixed pixel value (e.g., '800px'). Defaults to '100%'.
  • sliderHeight (string) - The height of the slider. Set it to your preferred height, such as '400px'. Defaults to '300px'.
  • imagesCount (number) - The number of images to display in the slider. This value is automatically set based on the loaded images.
  • divId (string) - An optional ID for the componentâ€™s root div element. Use this if you need to reference the slider programmatically.

JSON Configuration

Here is the structure of the JSON file you should provide to the fileToLoad input:


{
  "width": "800px",  
  "height": "400px", 
  "slideInterval": 3000, 
  "images": [
    {
      "src": "./assets/images/flag.jpeg",  
      "alt": "Image 1"  
    },
    {
      "src": "./assets/images/airplane.jpeg", 
      "alt": "Image 2"
    },
    {
      "src": "./assets/images/motorcycle.png", 
      "alt": "Image 3"
    }
  ]
}

How It Works

  • Images are automatically loaded from the provided JSON file.
  • The slider will start with the first image and automatically transition to the next image every few seconds based on the slideInterval value.
  • You can navigate between images manually using the previous and next buttons, or wait for the automatic transition.
  • Each image will be displayed according to the width and height defined in the JSON configuration or passed as input.
