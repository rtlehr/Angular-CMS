The Image Slider Component is a responsive and customizable carousel designed to display images dynamically. It fetches image data, including captions and additional details, from a JSON file, allowing for easy updates and configuration. The slider supports automatic transitions with a smooth animation effect, cycling through images from right to left after a set interval. Users can also navigate manually using previous and next buttons. The size of the slider is fully adjustable through configurable width and height properties, ensuring compatibility with various screen sizes and layouts. Built with Angular, Bootstrap, and custom CSS, this component blends functionality with a sleek, modern design.

JSON Code for the sample below 

    

{
    "width": "800px",
    "height": "400px",
    "slideInterval": 3000,
    "images": [
    {
        "src": "./assets/images/flag.jpeg",
        "alt": "Image 1"
    },
    {
        "src": "./assets/images/airplane.jpeg",
        "alt": "Image 2"
    },
    {
        "src": "./assets/images/motorcycle.png",
        "alt": "Image 3"
    }
    ]
}
