How to Use and Activate the Modal Window

The modal window can be triggered manually using a button or automatically when loading content. To manually open the modal, add a button with the data-click="openModalWindow" attribute and include optional JSON content in data-content.

Example: Manually Trigger Modal with <button>

      
  <button data-click="openModalWindow" data-content='{"title": "Hello", "content": "This modal was opened by a button!"}'>
    Open Modal
  </button>

 

Example: Manually Trigger Modal with <a>

      
  <a href="#" data-click="openModalWindow" data-content='{"title": "Hello", "content": "This modal was opened from a link!"}'>
  pen Modal from Link
  </a>
Open Modal from Link

 

To automatically trigger the modal when loading a content file, include a <trigger-modal> tag inside the HTML file being loaded. The modal will extract the data-content attribute and display the content.

Example: Auto-Trigger Modal on Load

      
  <trigger-modal data-content='{"title": "Auto Modal", "content": "This modal was triggered automatically!"}'></trigger-modal>

When the content containing <trigger-modal> is loaded, the modal will automatically display with the provided message.
