Image Display Component Instructions

The ImageDisplayComponent is used to display a gallery of images, with the option to view a larger version in a modal window when clicked. It handles both public and private images, with placeholders for private ones.

Usage

To use the ImageDisplayComponent in your Angular project:

  1. Import and configure the ImageDisplayComponent in your Angular module.
  2. Provide the component with the path to the JSON data file containing image information.
  3. Use the component in your HTML by passing the divId and the fileToLoad inputs as required.

Component Inputs

Input Name Type Description
fileToLoad String Path to the JSON file containing image data (e.g., 'assets/images-data.json').
divId String Unique ID for the div element containing the gallery of images.

JSON Data Structure

The component expects the following structure for the JSON data file:


[
    {
        "id": "1",
        "src": "./assets/images/flag.jpeg",
        "largeSrc": "./assets/images/flag.jpeg",
        "title": "Image 1",
        "description": "This is the description for Image 1.",
        "private": "false" 
    },
    {
        "id": "2",
        "src": "./assets/images/airplane.jpeg",
        "largeSrc": "./assets/images/airplane.jpeg",
        "title": "Image 2",
        "description": "This is the description for Image 2.",
        "private": "true"
    },
    {
        "id": "3",
        "src": "./assets/images/motorcycle.png",
        "largeSrc": "./assets/images/motorcycle.png",
        "title": "Image 3",
        "description": "This is the description for Image 3.",
        "private": "false"
    }
]

Features

  • Displays images in a grid layout.
  • Private images are replaced with a placeholder image.
  • Clicking an image opens a modal with a larger view and description.
  • Clicking the close button or anywhere outside the modal closes it.

Conclusion

This ImageDisplayComponent provides an easy way to create an interactive image gallery with modal previews, while also supporting both public and private images.
