NewsComponent Usage Guide

The NewsComponent is an Angular component used to display small bits of information, such as news items or blog posts. It retrieves the data from a provided JSON file and presents it in a list format.

1. Installation & Import

Ensure that you import the component into your Angular module or use it as a standalone component.


import { NewsComponent } from 'path-to-component/news.component';

2. Component Selector

Use the following selector to include the component in your Angular template:


<app-news [fileToLoad]="'news-data.json'" [divId]="'news-list'"></app-news>

3. Input Properties

Property Type Description
fileToLoad string The path to the JSON file that contains the news data.
divId string Sets the ID of the container div for the news list.

4. Features

  • Dynamic News Loading: Loads news data from an external JSON file.
  • News Item Layout: Each news item is displayed with an image, title, date, and content.
  • Responsive Design: The component adapts to different screen sizes.

5. Example Usage

To load a JSON file containing news data and display the list, include the component as shown below:


<app-news [fileToLoad]="'news-data.json'" [divId]="'news-list'"></app-news>

6. JSON Structure

The news data should follow this structure:


[
{
    "id": "1",
    "title": "Breaking News: Angular Rocks!",
    "image": "https://via.placeholder.com/150",
    "date": "December 11, 2024",
    "content": "Angular has released its latest version with amazing new features and improvements..."
},
{
    "id": "2",
    "title": "Tech Trends for 2024",
    "image": "https://via.placeholder.com/150",
    "date": "December 5, 2024",
    "content": "Explore the top tech trends of the year, including AI, blockchain, and more."
},
{
    "id": "3",
    "title": "Community Spotlight: Open Source Contributions",
    "image": "https://via.placeholder.com/150",
    "date": "November 30, 2024",
    "content": "Highlighting the most significant contributions from the open-source community this year."
}
]

7. News Item Layout

Each news item is displayed with the following structure:


<div class="list-group-item newsItem">
<div class="d-flex align-items-start">
    <img [src]="news.image" alt="News Image" class="me-3" style="width: 100px; height: auto; object-fit: cover;">
    <div>
        <h5 class="mb-1">{{ news.title }}</h5>
        <p class="text-muted mb-1">{{ news.date }}</p>
        <p class="mb-2">{{ news.content }}</p>
        <a href="#" class="btn btn-primary btn-sm">Read More</a>
    </div>
</div>
</div>

8. Error Handling

If the JSON file cannot be loaded, the component will log an error message to the console:


console.error('Error fetching JSON file:', error);

Conclusion

The NewsComponent provides a flexible way to display dynamic news data from a JSON file, with support for responsive design and easy customization.
