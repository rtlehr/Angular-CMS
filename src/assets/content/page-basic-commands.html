Basic Commands

Create a project

In the terminal navigate to the folder you want to create the project and enter


    ng new NAME-OF-PROJECT

Start the application

To start the application, in the termianal, navigate inside the application folder and enter


    ng serve

Create a new component

To create a new component enter the following


    ng generate component components/COMPONENT-NAME

    or

    ng g c components/COMPONENT-NAME

adding the "components/" before the COMPONENT-NAME will put components in a folder called, "components"
